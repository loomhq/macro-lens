Macro is an experiment that explores a simpler version of Lens. Macro is designed in the browser and uses advantages like relative units.

Principles

  • Declarative over imperative.
  • By default use native HTML components (like select).
  • Use relative units like rem and vw instead of px.

Features

  • Built with pure CSS.
  • Dynamic base unit value, instead of 8px or 0.5rem, the unit is calc(0.5vw + 0.5rem), this value is relative to the window size.
  • Dynamic type: the type sizes are relative to the window, the sizes will shrink/grow with the browser.
  • Dynamic spacing: the base unit is relative to the window, the spacing will shrink/grow with the browser.
  • New --textBox-padding variable used to define the padding of buttons, inputs and other similar elements.
  • Less code: this version has 221 lines of CSS.
